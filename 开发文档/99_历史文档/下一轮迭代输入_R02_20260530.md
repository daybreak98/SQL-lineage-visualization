实现 JSON 元数据导入功能，用户可以粘贴 JSON 或上传 JSON 文件将表字段元数据导入 SQLite。

1. POST /api/metadata/import/preview
2. POST /api/metadata/import/commit
3. JSON 校验（schema_version / 重复字段 / 结构完整性）
4. import_jobs / import_errors CRUD
5. 事务化导入 + 失败回滚
6. Golden Case GC-P0-008
