# 测试用例目录说明

本目录保存 SQL 血缘解析、数仓验数、AB 实验、搜索、优惠等业务 SQL 样例。

## 目录说明

| 目录 | 说明 |
|---|---|
| `00_血缘解析基准/` | 血缘解析基准文档与期望解析结果 |
| `数仓/` | 数仓验数与核心底表相关 SQL |
| `AB实验/`、`AB_rank/`、`AB实验_流量/`、`AB实验_定价/` 等 | AB 实验相关 SQL 样例 |
| `搜索/` | 搜索业务相关 SQL |
| `优惠先享需求/` | 优惠先享需求相关 DDL / SQL |
| `最终建表/` | 最终建表 SQL |
| `99_非业务元数据/` | IDE 元数据等非业务文件，不作为测试输入 |

## 使用规则

- 业务 SQL 样例可以作为复杂 SQL 回归来源。
- `00_血缘解析基准` 是血缘解析基准材料，优先用于验证输出结构。
- `99_非业务元数据` 不参与血缘解析和 Golden Case。
- 新增测试样例时，优先按业务主题放入已有目录；若是通用血缘基准，放入 `00_血缘解析基准`。
